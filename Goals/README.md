## âœï¸ How to Use This System

### At the Start of Each Quarter
1. Review last quarter's progress
2. Set 3-4 major objectives for next 3 months
3. Create `quarterly/YYYY-QX/objectives.md`
4. Align objectives with language learning phases

### At the Start of Each Month
1. Break down quarterly objectives into monthly sprint
2. Create `monthly/YYYY-MM/sprint.md`
3. Define specific deliverables and metrics
4. Link to language checklists

### At the Start of Each Week
1. Review monthly sprint progress
2. Create `weekly/YYYY-WXX/focus.md`
3. Define 3-5 key tasks for the week
4. Block time for each language family

### Daily
1. Check weekly focus in morning
2. Work on language-specific tasks
3. Update [journal](../journal/) in evening
4. Note progress toward weekly/monthly/quarterly goals
